# **App Name**: Cricket IQ

## Core Features:

- Player Rating Input: Enable selectors to rate players based on batting, bowling, wicket-keeping, and fielding skills using a 0-5 scale.
- Player Categorization: Allow categorization of players by batting order (top, middle, low) and bowling style (fast, medium, off-spin, leg-spin), plus dominant hand.
- Performance Display: Display player ratings and categories to selectors, players, and parents.
- Team Composition Tool: Team composition tool: Generate a ranked list of players based on a defined team composition (e.g., 2 wicketkeepers, 4 fast bowlers, 2 spinners). This will use AI to assess the constraints specified by the selection committe, assess which positions need to be filled, then fill them based on ranking within each applicable position.
- Average Score Calculation: Present data on average player scores, enabling comparisons based on the score / games played.

## Style Guidelines:

- Primary color: Dark moss green (#4A6150) to evoke a sense of tradition, stability, and nature.
- Background color: Very light moss green (#F1F4F2), complementing the primary and providing a gentle backdrop.
- Accent color: Russet brown (#805533), an analogous hue that adds sophistication.
- Body and headline font: 'PT Sans', a modern yet accessible sans-serif for clear data presentation and readability.
- Use icons representing batting, bowling, and fielding to visually differentiate player stats.
- Ensure a clean, data-focused layout prioritizing clarity and ease of use for quick player assessments.
- Subtle animations when displaying performance graphs
